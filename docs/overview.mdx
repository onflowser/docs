# What is Flowser?

[Flowser](https://flowser.dev) is first-of-its-kind development tool for [Flow blockchain](https://flow.com). It does the heavy work of managing blockchain emulator, inspecting the current state, interacting with the network and much more!

:::

## Features

- View complete state of your local blockchain, including:
    - Accounts (aka. wallets)
    - Blocks
    - Transactions
    - Contracts
    - Events
- [Account storage](https://developers.flow.com/cadence/language/accounts#account-storage) inspector
- [Development wallet](https://github.com/onflow/fcl-dev-wallet)
- Execute transactions
- [State snapshots](https://github.com/onflow/flow-emulator#managing-emulator-state)
- View emulator logs
